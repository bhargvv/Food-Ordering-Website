# Admin dashboard

React + Vite app for managing the food delivery backend: orders, menu, and related operations.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Dev server with HMR |
| `npm run build` | Production build to `dist/` |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | ESLint |

## API URL

The panel talks to the backend at `http://localhost:4000` (see `src/App.jsx` and `src/assets/assets.js`). Change these if the API is on another host or port.

## Prerequisites

- Node.js 18+
- Backend running (see [../README.md](../README.md))
